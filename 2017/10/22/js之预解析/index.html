规则

js解析器在运行js代码的时候 分为两步：

第一步 把所有的 变量、函数、参数 提前到当前作用域的顶部。
第二步 逐行解读代码 从左到右、从上至下。
表达式：= + - * / % ++ – ! 参数 …
